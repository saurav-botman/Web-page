Programme Overview

This 3-year programme focuses on software design, development, and project management. All modules are 30 credits unless stated otherwise.

Year 1 (Level 4)

Core Modules

  • Introduction to Programming (30 credits)
  • Computer Systems Fundamentals (30 credits)
  • Mathematics for Computing (30 credits)
  • Web Technologies (30 credits)
  • Academic Skills (0 credits)

Year 2 (Level 5)

Core Modules

  • Software Engineering Principles (30 credits)
  • Object-Oriented Programming (30 credits)
  • Database Systems (30 credits)
  • Software Testing and Quality Assurance (30 credits)
  • Employability Skills (0 credits)

Year 3 (Level 6)

Core Modules

  • Advanced Software Development (30 credits)
  • Distributed Systems (30 credits)
  • Final Year Project (30 credits)
  • Professional Practice (0 credits)

Elective Modules (Choose 30 credits)

  • Mobile Application Development (30 credits)
  • Cloud Computing (30 credits)
  • Cyber Security Fundamentals (30 credits)
  • Artificial Intelligence Basics (30 credits)
