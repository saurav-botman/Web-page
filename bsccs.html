Programme Overview

This 3-year programme is semester-based. All modules are 30 credits unless stated otherwise.

Year 1 (Level 4)

Core Modules

  • Database Design and Implementation (30 credits)
  • Fundamental Concepts of Computer Science (30 credits)
  • Computer Programming (30 credits)
  • Operating Systems and Networks (30 credits)
  • Academic Skills and Employability (0 credits)

Year 2 (Level 5)

Core Modules

  • Object Oriented Design and Development (30 credits)
  • Data Structures and Algorithms (30 credits)
  • Web Application Development (30 credits)
  • Agile Development Team Project (30 credits)
  • Employability (0 credits)

Year 3 (Level 6)

Core Modules

  • Software Development: Methods & Standards (30 credits)
  • Big Data and Machine Learning (30 credits)
  • Development Project (30 credits)
  • Philosophy of Computer Science (0 credits)

Elective Modules (Choose 30 credits)

  • Functional Programming (30 credits)
  • Advanced Web Development (30 credits)
  • Fuzzy Logic and Inference Systems (30 credits)
  • Privacy and Data Protection (30 credits)
