Programme Overview

This undergraduate programme teaches students to analyze data using statistical methods, machine learning, and big data technologies. Students gain practical programming and data analysis skills.

Year 1

  • Introduction to Programming
  • Mathematics for Data Science
  • Fundamentals of Data Analysis

Year 2

  • Data Structures and Algorithms
  • Machine Learning
  • Database Systems
  • Data Visualisation

Year 3

  • Big Data Analytics
  • Artificial Intelligence
  • Data Mining
  • Final Year Project
