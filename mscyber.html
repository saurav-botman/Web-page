Programme Overview

This postgraduate programme prepares students to protect computer systems, networks, and data from cyber threats. Students gain expertise in ethical hacking, cryptography, digital forensics, and security management.

Core Modules

  • Network Security and Cryptography
  • Secure Software and Systems
  • Digital Forensics
  • Security Management and Governance
  • Penetration Testing & Ethical Hacking

MSc Project

Research or industry-based project in Cyber Security.

Optional Modules

  • Machine Learning for Cyber Security
  • Advanced Cryptography
  • Distributed Systems
